# Language Hub Jeopardy — Units 7–9 (Mind · Art · Money)

An interactive, self-contained Jeopardy-style revision game for CEFR **B1** English
(Macmillan *Language Hub*, Units 7–9). Vocabulary and grammar questions with a
clickable board, team scoring, reveal-on-click answers, and selectable colour themes.

## Play it

Open **`index.html`** in any browser — that's it. No server, no internet, no install.
When hosted on GitHub Pages, the game is live at the project's Pages URL.

## Features

- 3 unit boards × 6 categories × 5 point tiles (100–500) = **90 questions**.
- Team scoreboard (2–6 teams, editable names), award points on a correct answer.
- Reveal-on-click answers; played tiles are crossed off.
- 6 colour themes + a custom colour picker.
- Progress and scores are saved per browser (`localStorage`).

*Questions are original revision material; the source coursebook is not included.*
